6 Inserting columns based on overlap

If you want to insert columns from one PyRange, based on where it overlaps with another, you can use the insert method.

It takes the arguments strandedness ("same", "opposite" or False/None), how ("first", "containment", False/None), suffix and overlap_only.

If the overlap_only-flag is True, the only rows in the PyRange which are kept are those where the insertion and insertee-PyRanges overlap.

import pyranges as pr
exons, cpg = pr.data.exons(), pr.data.cpg()
print(cpg)
