16 Joining Ranges

You can combine all the intervals that overlap in two PyRanges objects with the join method. If you do not use a suffix, the default _b is chosen.

import pyranges as pr
gr = pr.data.aorta()
gr2 = pr.data.aorta2()
print(gr.join(gr2, suffix="_2"))

The join method also takes a how argument, which currently accepts the option "containment", "inner" (default), "outer", "left" and “right”. Containment requires that the intervals in self be completely within the intervals in other. The others are similar to SQL-style inner, outer, left and right joins.

f1 = pr.data.f1()
f2 = pr.data.f2()
print(f2.join(f1, how="containment"))
## +--------------+-----------+-----------+------------+-----------+--------------+-----------+-----------+------------+-----------+--------------+

After joining, you can use the method new_position() to find the union or intersection of the joined ranges. The default suffixes are ["_a", "_b"], but a suffixes argument overrides this.

Positions with the second suffix should exist in the PyRanges already. The first suffix is used to rename the old Starts and Ends so they are kept.

print(f2.join(f1).new_position("intersection"))
## +--------------+-----------+-----------+-----------+-----------+------------+-----------+--------------+-----------+-----------+------------+-----------+--------------+
## | Chromosome   |     Start |       End |   Start_a |     End_a | Name       |     Score | Strand       |   Start_b |     End_b | Name_b     |   Score_b | Strand_b     |
## | (category)   |   (int32) |   (int32) |   (int32) |   (int32) | (object)   |   (int64) | (category)   |   (int32) |   (int32) | (object)   |   (int64) | (category)   |

If you want to swap which columns are considered the Start and End, you can use the swap argument.

In this case, the suffixes should be the suffixes of the columns you want to swap. So if the columns are named “Start”, “End”, “Start_b” and “End_b”, you should give the suffixes ["“,”_b"]

gr1, gr2 = pr.data.chipseq(), pr.data.chipseq_background()
j = gr1.join(gr2)
print(j)
